# 功能特色

GoWherer 提供了一组围绕“记录旅程”设计的核心能力，帮助你在出行时低成本记录、回顾并分享完整轨迹。

## 核心功能

### 旅程管理
- 开始/结束旅程：通过简单操作管理一次完整出行。
- 历史记录：按时间快速浏览历史旅程。
- 标签体系：为旅程和条目打标签，便于后续检索。

### 时间线记录
- 多种记录类型：支持文字、位置、照片和视频。
- 媒体导入：可从系统相册选择图片与视频。
- 条目编辑：支持对已记录内容进行修改或删除。

### 地理能力
- 实时定位：自动获取当前位置。
- 地点名称解析：将经纬度转换为可读地址。
- 坐标转换：支持常见坐标系转换场景。

### 地图与路线
- 原生地图体验：移动端具备流畅地图交互。
- Web 地图兜底：在网页端自动启用在线地图。
- 轨迹展示：按时间顺序可视化行程路径。

### 统计与导出
- 行程统计：自动计算距离、时长、平均速度等指标。
- PDF 导出：导出路线与统计摘要，适合存档和分享。

### 主题与体验
- 深色模式：支持浅色/深色主题切换。
- 偏好持久化：自动记住主题选择。
- 交互动效：提供平滑、自然的页面反馈。

# Features

GoWherer provides a set of core capabilities designed around "recording journeys" to help you capture, review, and share your complete trajectory at low cost while traveling.

## Core Features

### Journey Management
- Start/End Journey: Manage a complete trip with simple operations.
- History Records: Quickly browse historical journeys by time.
- Tag System: Tag journeys and entries for easy retrieval.

### Timeline Recording
- Multiple Record Types: Support text, locations, photos, and videos.
- Media Import: Select images and videos from system albums.
- Entry Editing: Support modifying or deleting recorded content.

### Geographic Capabilities
- Real-time Positioning: Automatically get current location.
- Place Name Resolution: Convert coordinates to readable addresses.
- Coordinate Conversion: Support common coordinate system conversion scenarios.

### Maps & Routes
- Native Map Experience: Smooth map interaction on mobile devices.
- Web Map Fallback: Automatically enable online maps on web pages.
- Trajectory Display: Visualize travel paths in chronological order.

### Statistics & Export
- Trip Statistics: Automatically calculate distance, duration, average speed, and other metrics.
- PDF Export: Export route and summary statistics, suitable for archiving and sharing.

### Theme & Experience
- Dark Mode: Support light/dark theme switching.
- Preference Persistence: Automatically remember theme selection.
- Interactive Animations: Provide smooth and natural page feedback.

features:
    - title: 快速记录
      details: 开始旅程后，一键记录文字、位置、照片、视频和音频，让每个瞬间都不会被遗漏。
    - title: 模板记录
      details: 内置"出发"、"到达"、"休息"、"打卡"模板，一键录入预设文本与标签，支持旅行与通勤双模式。
    - title: 智能定位
      details: 自动获取位置信息，通过高德逆地理编码将经纬度转换为可读地点名称。
    - title: 坐标兼容
      details: 内置 WGS84 与 GCJ02 坐标系互转算法，无缝对接高德地图 SDK 与国际坐标系数据。
    - title: 路线可视化
      details: 在地图上直观展示旅程轨迹，Web 端自动切换在线地图，保证跨平台体验。
    - title: 数据统计
      details: 自动统计总距离、总时长、平均速度等关键指标，帮助你回顾每次出行。
    - title: PDF 导出
      details: 一键导出包含路线预览图和统计信息的 PDF，便于分享与存档。
    - title: 深色模式
      details: 支持浅色/深色主题切换，并可跟随系统自动适配，偏好设置自动保存。
    - title: 本地日志
      details: 应用内日志系统，支持导出日志文件用于错误追踪与问题排查。
